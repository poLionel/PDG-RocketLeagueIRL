üéÆ Controls Help

Car Controls:

  • Forward/Backward/Stop: Use the direction buttons
  • Steering: Use the slider or steering buttons
  • Boost: Toggle boost on/off while moving
  • Car Selection: Select a car to control it exclusively

Game Management:

  • Always Available: A game is always ready - no need to create one
  • Start/Reset Game: Begins a new match and resets all scores/time
  • Pause Game: Stops the timer but preserves game state
  • Resume Game: Continues from where the game was paused
  • End Game: Permanently finishes the current match
  • Auto-End: Game automatically ends when time expires
  • Team Management: Add teams and assign cars to teams
  • Scoring: Record goals for teams (red, blue, green, yellow)

API Testing Features:

  • Always-Available Game: No create_game needed - game is always ready
  • start_game: Resets and starts a new match (with optional match length)
  • stop_game: Pauses the game (preserves state and time)
  • resume_game: New action to continue paused games
  • end_game: Permanently ends the current match
  • goal_scored: Action that matches documentation
  • Auto-Timer: Game automatically ends when time expires

Keyboard Controls:

  • W / ‚Üë: Move forward
  • S / ‚Üì: Move backward
  • A / ‚Üê: Steer left (incremental)
  • D / ‚Üí: Steer right (incremental)
  • Space / Esc: Emergency stop
  • Shift: Activate boost

Test Scenarios:

  • Simulate Full Match: Complete game flow with pause/resume testing
  • Quick Goal Test: Rapid goal scoring simulation
  • Game Flow Test: Tests start/pause/resume/end cycle
  • Emergency Stop: Stops both game and all car movement

Note: Select "Keyboard Control" on a car to enable keyboard controls for that specific car.
